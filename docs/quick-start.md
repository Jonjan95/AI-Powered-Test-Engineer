# Quick Start

## Backend

### Windows

```powershell
cd backend
.\mvnw.cmd spring-boot:run
```

### Mac

```bash
cd backend
./mvnw spring-boot:run
```

Health endpoint:

http://localhost:8080/api/health

---

## Frontend

```bash
cd frontend
npm run dev
```

Frontend:

http://localhost:3000

---

## Git

Before starting work:

```bash
git pull
```

After finishing work:

```bash
git add .
git commit -m "description"
git push
```
